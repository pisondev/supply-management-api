# Supply Management API - MBG (Subtema 1)

Pengerjaan Penugasan Backend OTI Pertemuan 4: Sprint 1  (Containerized API).

## Tech Stack

- **Language:** Go (1.25.3)
- **Framework:** Go Fiber (v2)
- **ORM:** GORM
- **Database:** PostgreSQL 16
- **Infrastructure:** Docker & Docker Compose
- **Validation:** Go Playground Validator (v10)
